## About
Modified copy of browser-based Ising Model in Javascript from: https://github.com/mattbierbaum/ising.js

## Usage
```
# Running
$ docker run -d --rm --name ising -p 8080:80 tigerj/ising

# Stopping
$ docker stop ising

# Starting
$ docker start ising

# Removing
$ docker rm ising
```

Navigate to: http://localhost:8080
